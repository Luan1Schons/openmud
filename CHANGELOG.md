All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Sistema de respawn de monstros configurável
- Sistema de geração de quests com IA (OpenAI/Anthropic)
- Sistema de dungeons
- Sistema de classes e raças
- Sistema de magia e feitiços
- Sistema de lojas e comércio
- Suporte a múltiplos mundos
- Criação de personagens personalizada
- Sistema de autenticação

### Changed
- Melhorias na organização do projeto
- Documentação reorganizada na pasta `docs/`
- README atualizado com badges e melhor estrutura

## [0.1.0] - 2024-01-XX

### Added
- Versão inicial do MUD
- Sistema básico de salas e navegação
- Sistema de chat entre jogadores
- Suporte para TinTin++ e outros clientes MUD
- Sistema de cores ANSI
- Banco de dados SQLite para persistência
- Sistema básico de NPCs
- Sistema básico de monstros
- Sistema básico de quests
